# Entendendo Factory Method

Este repositório foi criado exclusivamente para fins de aprendizado do padrão de projeto **Factory Method**. Aqui você encontrará exemplos, experimentos e anotações feitos durante o estudo do padrão — o objetivo principal é entender como funciona a criação de objetos delegada a fábricas.

## Sobre
O repositório não é um produto pronto nem uma biblioteca de produção. Foi montado para:
- Demonstrar o conceito do Factory Method;
- Registrar variações e experimentos com implementações;
- Servir como material de estudo e referência pessoal.

## Conteúdo
- Exemplo(s) de implementação do Factory Method em linguagem JAVA;
- Notas e comentários explicativos;
